🚗 CarSure

📌 Om projektet

CarSure är en webbaserad plattform som kombinerar bilförsäljning med försäkringstjänster. Målet är att förenkla processen för både köpare och säljare genom att erbjuda en helhetslösning där man kan köpa, sälja och försäkra bilar på ett och samma ställe.

Projektet är en del av ett examensarbete med fokus på modern webbutveckling och användarvänlig design.

🎯 Syfte

Syftet med CarSure är att:

Förenkla bilköp och bilförsäljning
Integrera försäkring direkt i köpprocessen
Skapa en trygg och smidig användarupplevelse
Kunna konkurrera med befintliga plattformar som Blocket
⚙️ Funktioner
🔍 Söka och filtrera bilar
🚗 Lägga upp annonser för att sälja bil
❤️ Spara favoritbilar
📊 Visa detaljerad information om fordon
🛡️ Visa och jämföra försäkringar
📅 Boka möte med försäkringsrådgivare
💳 Möjlighet att köpa bil med försäkring direkt
🛠️ Tekniker

Projektet är byggt med:

C# / .NET (MVC)
HTML / CSS / JavaScript
Git & GitHub för versionshantering
Visual Studio Code som utvecklingsmiljö
Databas? sqlite?/mongodb?
